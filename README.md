# 🍔 GAULA-KORNER — Full Stack Food Ordering Web App

GAULA-KORNER is a modern, responsive, mobile-first food ordering application built to showcase a beautiful interactive menu, smooth animations, and a clean shopping cart experience. The project includes a Node.js/Express backend serving product data, and a Vite + React + Zustand frontend for state management.

---

## Features

### **Frontend**

- Built with **React + Vite**
- Smooth **fall-in animation effects** using **Framer Motion v11**
- Fully responsive menu layout (mobile, tablet, desktop)
- Animated Add-to-Cart button (with success checkmark)
- Organized display by category (Wings, Burgers, Chips, Wraps, Drinks)
- Fast rendering via **Zustand global store**
- High-quality Unsplash product images

### **Backend**

- Node.js + Express server
- `/api/products` endpoint serving product data
- Data loaded securely from `products.json`
- CORS enabled
- Supports expansion to checkout/payments, auth, etc.

### **Additional Features**

- Beautiful UI animations
- Category filtering
- Persistent layout
- Smooth image hover zoom effect
- Ready for deployment (Vercel, Netlify, Render, Railway)
